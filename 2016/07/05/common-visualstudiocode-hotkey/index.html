通用 - Visual Studio Code－常用熱鍵

發表於2016-07-05|更新於2022-07-14|其它通用
|字數總計:105|閱讀時長:1分鐘|閱讀量:

說明

由於之前常用的 Sublime 多行編輯實在太方便，所以就找了一下 Visual Studio Code 是否有支援相關功能

多行編輯

Shift+Alt+Left

選擇相同詞

Ctrl+D

代碼整理

Shift+Alt+F

代碼開闔

全開|Ctrl+Alt+Shift+]
全瞌|Ctrl+Alt+Shift+[
單開|Ctrl+Shift+]
單闔|Ctrl+Shift+[

快速換行

Alt+↑ or Alt+↓

復製整行

Shift+Alt+↑ or Shift+Alt+↓
